MINI PROJECT

  • Each team can have 2-4 participants.
  • Participants must make a prototype of their project.
  • The prototype must be displayed and presented with an effective power point presentation.
  • The presentation should contain a simple description about their project.
  • Following points can be used to describe the project in the presentation:
    • Introduction
    • Objective
    • Block Diagram
    • Expected Outcomes
    • Application
  • Teams will be given a space to set up their model at the venue for exhibition and judging.
  • A total of 15 minutes will be given for setup and presentation of the model.
